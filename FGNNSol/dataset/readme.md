## Dataset Description
Here, we provide the raw data for the training set, test set, and validation set (located in the `csvFile` directory), as well as protein sequence files (".fasta" files), protein 3D structure data (".pdb" files), and files with all extracted features (".pkl" files). You can find them in the corresponding folders.

Among these, due to the large size of the .pkl files for the training set, we have uploaded them to Google Drive. You can click the link to access them: https://drive.google.com/drive/folders/11-Zxt0rXgfxb635Yz043xhVP7FMlrTni?usp=drive_link
